- 修学旅行の目標 -

『三日間で古都の歴史を学び、みんなで3150サイコーの思い出にしよう。』

- おすすめ観光スポット -

奈良公園

春日大社

1300年の歴史を誇る春日大社は、平城京の守護と国民の繁栄を祈願するために神護景雲じんごけいうん2年（768）年に創られた神社。

興福寺

寺院内には国宝に指定されている阿修羅像や、その他に数多くの寺宝じほうを所蔵しょぞう。

浮雲園地

奈良公園最大の芝生広場でもあり、奈良公園内にある広々とした芝生が広がる奈良公園有数の憩いの場。少し休憩を取るのもいい。

奈良国立博物館

「東京国立博物館」に次いで日本で2番目に古い国立博物館。13の展示室がある。

若草山

奈良市内で最も有名な名勝地で、標高342メートルの小さな山。奈良市内が一望でき、奈良一の眺望と言われている。

正倉院

東大寺を創った聖武天皇の遺品をはじめとした宝物が保管されている校倉造あぜくらづくりの大規模な高床式倉庫。

嵐山付近

渡月橋

桂川かつらがわにかかる「渡月橋」は、嵐山のシンボルといえる存在。

竹林の小径

野宮ののみや神社から大河内山荘おおこうちさんそうていえんを結ぶ「竹林の小径ちくりんのこみち」は、約400mにわたって竹林が続く幻想的な場所。

天龍寺

「天龍寺」は、足利尊氏を開基とし、1339年に創建された寺院。

嵯峨野トロッコ列車

崖の上を走るトロッコ列車の車窓から自然豊かな嵯峨野さがのの風景を味わえる観光鉄道。

保津川下り

亀岡から嵯峨嵐山さがあらしやまへと流れる保津川ほつがわを舟に乗って下るアクティビティ。

キモノフォレスト

艶やかな友禅ゆうぜんを用いたポールが約600本も並ぶ光景は美麗びれいで、京の風情たっぷり。
